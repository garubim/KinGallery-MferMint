# KinGallery + MferMint

A minimal Next.js app focused on OnchainKit + paymaster flow for minting art on Base.

## Quick Start

1. Clone the repo
2. `npm install`
3. Set environment variables (see .env.example)
4. `npm run dev`

## Features

- Magic mint button with animated phrases
- Paymaster sponsorship for gas-free minting
- Farcaster mini app manifest
- Wallet Server v2 integration

## Deployment

Deploy to Netlify with functions for paymaster relay.
